# Oil & Gas Data Portfolio

This repository contains selected projects and case studies related to oil & gas data analysis and engineering applications.

The work includes drilling, production, and reservoir-focused projects using data tools and petroleum engineering software.

---

## 📂 Project Categories

### Drilling

* MWD data analysis
* Mud system analysis

### Production

* Water cut and WOR analysis
* Production data evaluation

### Reservoir

* EOR screening
* Reservoir simulation

### Engineering Projects

* Artificial lift systems
* Injection methods (gas, water, polymer)

---

## 🧠 Tools Used

* Python, SQL, Excel
* OFM, Petrel, PIPESIM
* Data analysis and engineering workflows

---

## 🎯 Purpose

This portfolio demonstrates practical application of data engineering and petroleum engineering concepts to real-world oil & gas problems.
